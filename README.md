# pGOLD - 数字黄金生态系统

## 项目简介

pGOLD是一个由PAXG储备支撑的数字黄金代币生态系统，提供：
- 💰 1克黄金代币化，门槛降低31倍
- 🔄 去中心化交易所（Swap）
- 📱 完整的DApp应用
- 📊 五轨激励机制
